Monolithe modulaire

Complexité

  • Essentielle : le problème que l'on cherche à résoudre
  • Obligatoire : la réponse approtée à ce problème
  • Accidentelle : arrivée on ne sait comment, utile à on ne sait quoi ...

Couplage

C4 Model

  • Context : système applicatif
  • Container : unité de déploiement
  • Component : blocs structurels majeurs
  • Class : implémentation

Les choix

  • DDD => isolation des contextes
  • CQRS => isolation lecture / écriture
  • Séparation du code en bundles pour permettre la création des conteneurs
